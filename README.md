# duke
Java's mascot and an enhanced entry-point exploration.

## Usage

Use `Duke.java` as a single-file source-code program.

```shell
java Duke.java build
```

Use `Duke.java` interactively in a JShell session:

```shell
jshell Duke.java
...
> var duke = new Duke.Program()
> duke.build()
...
> /exit
```
